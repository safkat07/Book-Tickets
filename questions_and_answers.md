###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>After the forEach loop, the count variable will contain the number of elements in the nums array that are not equal to 0. In this case, there are three such elements (1, 2, and 3), so count will be 3, and console.log(count) will print 3</i>

</p>
</details>
